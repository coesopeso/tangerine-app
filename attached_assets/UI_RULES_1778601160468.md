\# DASHBOARD\
\
\# Filosofia\
\
La dashboard NON deve:\
- sembrare un gestionale anni 2000\
- essere troppo tecnica\
- avere troppo testo\
\
Deve:\
- mostrare overview rapida\
- essere leggibile in 5 secondi\
- evidenziare KPI importanti\
\
\-\--\
\

- SAVING RATE\
- FATTURATO\
- TASSE\
- SECCHIELLI\
\
\-\--\
\
