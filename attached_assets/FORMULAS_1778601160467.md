# 📊 GENNAIO — K CELLS

# K91 — Entrate Private

## Scopo

Somma tutte le entrate private non tassate.

---

## Formula

```excel
=SUMIF(D15:D28,"ENTRATA PRIVATA (no tasse)",C15:C28)
```

---

## Logica

Conta:
- solo entrate private
- non soggette a tasse
- non soggette a INPS

---

# K92 — Fatturato P.IVA INCASSATO

## Scopo

Somma fatture P.IVA realmente incassate.

---

## Formula

```excel
=SUMPRODUCT((D15:D28="FATTURA P.IVA")*(H15:H28="INCASSATO")*C15:C28)
```

---

## Logica

Conta SOLO:
- FATTURA P.IVA
- stato INCASSATO

NON conta:
- FATTURATO
- PROGRAMMATO

---

# K93 — Tuo Netto P.IVA

## Scopo

Calcolare il netto reale P.IVA dopo tasse e split socio.

---

## Formula

```excel
=SUMPRODUCT(
(D15:D28="FATTURA P.IVA")*
(H15:H28="INCASSATO")*
IF(E15:E28="SI",
(C15:C28-(C15:C28*SETUP!$C$4*SETUP!$C$5)-(C15:C28*SETUP!$C$4*SETUP!$C$11))/2,
C15:C28-(C15:C28*SETUP!$C$4*SETUP!$C$5)
))
```

---

## Logica

### SOCIO = NO

```txt
lordo - tasse
```

---

### SOCIO = SI

```txt
(lordo - tasse - INPS socio) / 2
```

# 🪙 CRYPTO — Formule GOOGLEFINANCE

## Prezzo live BTC
```excel
=GOOGLEFINANCE("CURRENCY:BTCUSD")
```

## Prezzo live ETH
```excel
=GOOGLEFINANCE("CURRENCY:ETHUSD")
```

## Valore attuale (per ogni asset)
```excel
=C{row}*F{row}
```
dove C = quantità, F = prezzo live

## PNL ($)
```excel
=G{row}-E{row}
```
dove G = valore attuale, E = investito

## PNL %
```excel
=IFERROR(H{row}/E{row},0)
```

=GOOGLEFINANCE(C{row},"price")
